Black Ops 6 Loader

Full setup guide — follow every step in order.

Undetectedv3.2.1
Before you start — Disable Windows Defender real-time protection. The loader triggers false positives.
Prerequisites
1

Disable Windows Defender

Windows Security → Virus & threat protection → Manage settings → Turn off Real-time protection. Add your Downloads folder as an exclusion.

2

Check Requirements

Make sure everything below is installed:

  • Windows 10/11 64-bit
  • Visual C++ Redistributables (latest)
  • .NET Framework 4.8+
  • Game updated to latest version
Installation
3

Download & Extract

Grab the loader from the Downloads page. Extract the ZIP to a folder on your desktop — not Program Files.

C:\Users\YourName\Desktop\BO6-Loader\
4

Run as Administrator

Right-click Loader.exe → Run as administrator. A console window will open.

5

Enter License Key

Paste your key from the purchase email or My Keys page.

XORA-XXXX-XXXX-XXXX-XXXX
6

Launch the Game

When the loader says "Waiting for game...", open BO6 through Steam or Battle.net. It auto-injects.

7

Open Menu

Press INSERT in-game to open the cheat menu. Configure and play.

Tip: Hit "Save Config" in the menu to keep your settings between sessions.
Troubleshooting
Defender is probably still on. Double-check real-time protection is off. Also make sure you're running as admin and no overlay software is interfering.
Keys are case-sensitive — check for typos. If it still fails, your key may be expired or HWID-locked to another machine. Contact support on Discord.
Verify your game files through Steam/Battle.net. Make sure the game is fully updated. Try launching the loader before the game.

HWID Spoofer

Universal spoofer — run this before any loader if you've been banned.

Undetectedv5.0.2
Important: Always run the spoofer BEFORE any game or loader. Re-run after every PC restart.
Steps
1

Disable Everything

Disable Defender, Secure Boot (BIOS if needed), and any third-party antivirus. The spoofer needs kernel access.

2

Download & Extract

Download from the Downloads page. Extract to desktop.

C:\Users\YourName\Desktop\HWID-Spoofer\
3

Run as Administrator

Right-click Spoofer.exe → Run as administrator. It'll show your current hardware IDs.

4

Enter Key & Spoof

Enter your license key. Click "Spoof All" and wait for the confirmation.

5

Restart if Prompted

Some setups require a reboot. If prompted, restart then launch your loader as normal.

Remember: The spoof resets on reboot. Always re-run before each session.
Troubleshooting
Disable Secure Boot in BIOS. Make sure no anti-cheat (EAC, BattlEye, Vanguard) is running.
You need a new game account. The spoofer changes your HWID so the new account won't be linked, but the old account stays banned.

Rainbow Six Loader

R6 Siege setup guide.

Undetectedv4.1.2
Before you start — Disable antivirus. R6 uses BattlEye — no other cheats can be running.
Steps
1

Disable Antivirus

Turn off Defender. Add the loader folder as an exclusion.

2

Download & Extract

Download and extract to desktop.

C:\Users\YourName\Desktop\R6-Loader\
3

Run Loader FIRST

Right-click Loader.exe → Run as administrator. Enter key. Do not launch the game yet.

4

Launch R6 Siege

Once the loader shows "Ready", open R6 through Ubisoft Connect. Press INSERT in-game.

Important: Always start the loader before the game. BattlEye blocks injection if the game loads first.
Troubleshooting
Loader must be running before the game. If you launched the game first, close everything and start over.
Recommended. Run the HWID spoofer before the loader for extra protection in Ranked.
